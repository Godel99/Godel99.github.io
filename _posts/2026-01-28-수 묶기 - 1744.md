[문제 링크](https://www.acmicpc.net/problem/1744) 

## **풀이**
### 접근
두 수의 곱과 그 합이 최대가 되려면 일단 수열이 **정렬**되어 있어야 **그리디**하게 풀 수 있습니다. 문제는 **양수, 음수, 0, 1 각각에 대해서 언제 합과 곱이 최선**인지 분기가 갈린다는 점입니다.

### 분기
1. **양수의 경우**
   * 두 양수 간에는 큰 것끼리 곱해주면 항상 더 큽니다. 내림차순 정렬을 합니다.
2. **1의 경우**
   * 1은 곱하면 손해이므로 더해줍니다.
3. **음수의 경우**
   * 두 음수 간에는 작은 것끼리 곱해주면 항상 더 큽니다. 오름차순 정렬을 합니다.
4. **0의 경우**
   * 0은 음수와의 곱에선 항상 0이고, 아닐 경우는 그냥 +0을 하면 되므로 음수와 묶어서 계산합니다.

```python
import sys
def print(*args, sep=" ", end="\n"): sys.stdout.write(sep.join(map(str, args)) + end)
def input(): return sys.stdin.readline().rstrip()

def main():
    n = int(input())
    pos = []
    neg = []
    ans = 0
    for _ in range(n):
        x = int(input())
        if x <= 0: neg.append(x)
        elif x > 1: pos.append(x)
        else: ans += 1
    pos.sort(reverse=True)
    neg.sort()
    for i in range(0, len(pos), 2):
        if i + 1 < len(pos): ans += pos[i]*pos[i+1]
        else: ans += pos[i]
    for i in range(0, len(neg), 2):
        if i + 1 < len(neg): ans += neg[i]*neg[i+1]
        else: ans += neg[i]
    print(ans)
    return 0
if __name__ == '__main__':
    sys.exit(main())
```

#include<bits/stdc++.h>
using namespace std;
using ll=long long;

int main(){
    cin.tie(0); cout.tie(0); ios::sync_with_stdio(false);
    int n; cin >> n;
    vector<int> pos, neg;
    int ans = 0;
    for(int i = 0; i < n; i++){
        int x; cin >> x;
        if(x <= 0) neg.push_back(x);
        else if(x > 1) pos.push_back(x);
        else ans++;
    }
    sort(pos.begin(), pos.end(), greater<int>());
    sort(neg.begin(), neg.end()); 
    for(int i = 0; i < pos.size(); i += 2){
        if(i + 1 < pos.size()) ans += pos[i]*pos[i+1];
        else ans += pos[i];
    }
    for(int i = 0; i < neg.size(); i += 2){
        if(i + 1 < neg.size()) ans += neg[i]*neg[i+1];
        else ans += neg[i];
    }
    cout << ans;
    return 0;
}
